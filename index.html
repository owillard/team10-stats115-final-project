Gallstone Bayesian Analysis

Stats 115 Final Project

Olivia Willard, YuNing Chou, Allinn Chen, Mihir Borkar

Topic and Motivation

  • Gallstones are a common condition that can lead to serious complications
  • Many clinical and body composition variables may influence gallstone risk
  • It is unclear which predictors are most important

Goal: Identify key predictors of gallstones and estimate risk

Research Question

Initial Question: Are gallstone more prevelant among patients with comorbidities compared to those without comorbidities? Can comorbidity predict gallstone?

Refined Question: How do clinical and body composition variables influence the probability that a patient has gallstones?

Dataset

  • Source: UCI Machine Learning Repository
    • Contains clinical data collected from the Internal Medicine Outpatient Clinic of Ankara VN Medical Park Hospital in Turkey
    • Contains information on 319 patients, of whom 161 were diagnosed with gallstone disease
    • Contains 38 variables describing patient health and characteristics

Data Cleaning

  • Renamed variables for clarity
  • We converted variables to binary:
    • Gallstone (reserved coding)
      • 0 = does not have gallstones
      • 1 = has gallstones
    • Comorbidity
      • 0 = no comorbidities
      • 1 = at least one comorbidity

Model Choice

We used a Bayesian Logistic Regression model

Why?

  • Binary outcome: gallstones vs. no gallstones
  • Directly models probabilities
  • Incorporates prior information
  • Provides uncertainty estimates

\[ \text{logit}(p_i) = \beta_0 + \beta_1(\text{vita_d}_i) + \beta_2(\text{bone_mass}_i) + \beta_3(\text{crp}_i) + \beta_4(\text{icw}_i) + \beta_5(\text{hgb}_i) \]

  • Main predictors selected based on EDA findings: Vitamin D, bone mass, CRP, ICW, and HGB

Prior Model

# fit prior predictive model
prior_model <- stan_glm(
  formula = gallstone ~ vita_d + bone_mass + crp + icw + hgb,
  data = data,
  family = binomial(link = "logit"),
  prior_intercept = normal(
    location = qlogis(0.0605),
    scale = abs(qlogis(0.065) - qlogis(0.056)) / 4,
    autoscale = F
  ),
  prior = normal(location = 0, scale = 2.5, autoscale = T),
  chains = 4, iter = 10000, seed = 84735,
  prior_PD = T, refresh = F, cores = 16
)
  • Intercept:
    • Intercept prior based on baseline gallstone prevalence obtained from research (~6%)
  • Coefficients:
    • Weakly informative Normal priors centered at 0 for regression coefficients (0, 2.5)

Why?

  • Prevent extreme estimates
  • Still allow data to dominate

Model Checking and Results

  • Trace plots showed good chain mixing
  • R-hat values were close to 1 (convergence achieved)
  • Density plot shows consistent chains
  • Effective sample sizes were acceptable
  • Posterior predictive checks suggested the model captured the general pattern of the data

Conclusion:

  • Model is reliable

Posterior Model

# fitting the posterior model using the observed data
posterior_model <- update(prior_model, prior_PD = FALSE)

  • After specifying the priors, we updated the model using the observed data to obtain the posterior distribution.

  • This allowed us to estimate how each predictor was associated with the probability of gallstones while incorporating uncertainty.

Model Comparison

We have tested with 6 different models containing the different amount of predictors:

  • Model 1 (7 predictors): vita_d + bone_mass + crp + icw + hgb + hfa + gender
    • Accuracy about 81.82%
  • Model 2 (6 predictors): vita_d + bone_mass + crp + icw + hgb + hfa
    • Accuracy about 80.25%
  • Model 3 (5 predictors): vita_d + bone_mass + crp + icw + hgb
    • Accuracy about 79.62%
  • Model 4 (4 predictors): vita_d + bone_mass + crp + hgb
    • Accuracy about 78.99%
  • Model 5 (3 predictors): vita_d + bone_mass + crp
    • Accuracy about 79.62%
  • Model 6 (1 predictor): comorbid
    • Accuracy about 49.53%

Final Model Choice

We selected Model 3:

  • Predictors: Vitamin D, bone mass, CRP, ICW, HGB
  • Balanced predictive performance and simplicity
  • Similar performance to larger models with fewer predictors
  • Better sensitivity than the smaller 3-predictor model

Performance:

  • Overall accuracy: about 79–80%
  • Better at identifying patients without gallstones than patients with gallstones

Model Checking and Results

The model suggests:

  • Higher Vitamin D, bone mass, and HGB are associated with higher odds of gallstones
  • Higher CRP and ICW are associated with lower odds of gallstones

Because the posterior intervals did not include 0, these predictors appeared meaningfully related to gallstone status in this dataset.

Key Takeaways

  • Comorbidity alone was not a strong predictor in this dataset
  • A combination of clinical and body composition variables provided much better prediction
  • Bayesian logistic regression helped us estimate gallstone risk while clearly showing uncertainty
  • Bayesian approach provides:
    • Probabilities
    • Uncertainty quantification
    • Interpretability
  • EDA helped refine the question, and Bayesian modeling helped us answer it in a principled way

Who Benefits?

  • Researchers studying gallstone risk factors
  • Clinicians interested in screening-related variables (better risk assessment)
  • Patients can receive earlier detection
  • Future work on medical risk prediction using Bayesian models

Limitations

  • Limited dataset and sample context
  • Possible missing predictors
  • Comorbidity may have been too simplified after binarization
  • Some predictors may be correlated or reflect similar biological information
  • The model slightly underestimated the observed gallstone proportion
